# heim_content_machine

Content machine for the Heim project.

## PROTECTED FILES — DO NOT ACCESS

The file `.env` in this directory contains secret credentials. Under NO circumstances should any Claude Code worker, script, or process read, grep, cat, or otherwise inspect its contents. This is enforced by `.claude/settings.json` permissions.deny AND by Commander policy.

If you need an environment variable from `.env`, ask the Commander for the specific variable via a prompt — do not read the file yourself.
